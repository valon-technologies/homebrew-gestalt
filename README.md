# homebrew-gestalt

Homebrew tap for Gestalt.

## Install

```sh
brew tap valon-technologies/gestalt
brew install valon-technologies/gestalt/gestaltd
brew install valon-technologies/gestalt/gestalt
```

## Maintenance

The `Formula/` directory is seeded from `valon-technologies/gestalt` and kept in
sync by this repository's GitHub Actions workflow.
